\texttt{Chain Rule for Natural Logarithms}

If $g$ is differentiable, then the derivative of $h(x) = \ln\left(g(x)\right)$ is
\[
h'(x) = \frac{g'(x)}{g(x)}
\]

\begin{example}
Find the derivative of each.
\begin{enumerate}[(a)]
    \item $y = \ln(x^4-2x)$
    \item $h(x) = \ln\left(\sqrt{6x-1}\right)$
    \item $f(x) = \left(\ln(x)\right)^3$
\end{enumerate}
\end{example}
